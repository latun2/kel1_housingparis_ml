# HouseParis_ml
# Paris Housing Prices Prediction

Machine Learning project untuk memprediksi harga properti di Paris menggunakan beberapa algoritma regresi, yaitu Linear Regression, Decision Tree Regressor, dan Random Forest Regressor.

## Dataset
Paris Housing Prices Dataset dari Kaggle  
https://www.kaggle.com/datasets/syedaeman2212/paris-housing-prices

## Metode yang digunakan
- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor

## Tahapan
- Exploratory Data Analysis (EDA)
- Data preprocessing (encoding & scaling)
- Train-test split (80:20)
- Model training
- Evaluasi (MAE, MSE, RMSE, R² Score)

## Hasil
Model terbaik diperoleh dari Linear Regression dengan R² Score tertinggi.
